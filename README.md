## 开始使用

1. 克隆此仓库
2. 安装依赖：`npm install` 或 `yarn` 或 `pnpm install`
3. 复制 `.env.example` 文件到 `.env` 并更新值。
4. 运行开发服务器：`npm run dev` 或 `yarn dev` 或 `pnpm dev`
5. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)
6. 使用任意用户名和密码登录系统
